# Projection Overview

Projection modules build read models from local runtime data.

## Rules

- Pure functions only.
- No mutation of input records.
- No DOM or IndexedDB access.
- No browser globals.
- Linear processing over input collections.

## Read Models

- Dashboard metrics
- Goal progress
- Financial health
- Execution planning summary
- Notification summary
- Calendar summary
- Scheduler summary
- Automation summary
