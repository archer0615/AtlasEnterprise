# Domain Service Catalog

## Purpose
Defines stateless domain services that encapsulate business logic not naturally owned by a single Aggregate.

### Core Services
| Service | Responsibility |
|---|---|
| DecisionService | Recommendation orchestration |
| CashFlowService | Cash flow calculations |
| PortfolioService | Portfolio analysis |
| LoanService | Mortgage optimization |
| RetirementService | Retirement projections |
| ScenarioService | Scenario comparison |
| ScoringService | Decision scoring |
| ExplainabilityService | Explain decision rationale |
| RiskService | Risk evaluation |
| AllocationService | Asset allocation |
