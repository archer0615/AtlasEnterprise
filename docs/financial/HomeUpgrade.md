# Home Upgrade Model

## Requirements
- Preferred area: Banqiao / Xinpu
- If possible: Taipei city, non-mountain area
- Building age: within 15 years
- Elevator building
- 3 bedrooms
- Indoor space: 26 ping or above
- Parking preferred
- Price is market-driven, estimated around TWD 30M–33M+

## Decision Rules
- If a perfect property appears and market conditions are acceptable, buying is preferred.
- If equity assets fall sharply before 2031, delay up to around 3 years.
- After home upgrade, investment assets should ideally remain at least TWD 10M.
- Spouse contribution is modeled only as an event contribution amount.
