## 修改原因

- 依據 `docs/architecture/ADR-001-PWA-Architecture.md`，將 Atlas v1 文件同步為 PWA v1 Runtime、Browser Runtime、IndexedDB Runtime。
- 將舊有伺服器、資料庫伺服器、後端執行期、雲端執行期與網路端點描述改為 Future Cloud Architecture 或 Future Cloud Mapping。
- 保留 Repository Pattern，但將 Repository Adapter 語意同步為 IndexedDB Adapter。
- 將 Database Mapping 語意調整為 PWA Runtime Mapping / Future Cloud Mapping，保留未來雲端資料庫映射方向。
- 在受影響文件加入 ADR-001 PWA Runtime Alignment 提示，避免 v1 與未來雲端架構混淆。
