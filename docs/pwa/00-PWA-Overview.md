# PWA Overview

## Scope

Atlas v1 is an installable, offline-capable, single-user, local-first web application.

## Required Capabilities

- Installability
- Offline application shell
- Offline Domain and Engine execution
- IndexedDB persistence
- Encrypted backup and restore
- Schema migration
- Update notification
- GitHub Pages deployment
- Browser compatibility diagnostics

## Unsupported in v1

Cloud sync, multi-user collaboration, server notifications, secret API integrations, and server-side scheduled jobs.
