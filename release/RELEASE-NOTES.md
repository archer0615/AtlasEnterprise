# Release Notes

## Atlas Enterprise v1.0.0

Atlas Enterprise v1.0.0 is a local-first, offline-first, static-first financial planning runtime for GitHub Pages compatible deployment.

## Included

- Financial local runtime
- Goal and financial health runtime
- Scenario and decision runtime
- Recommendation and execution planning runtime
- Notification, automation, scheduler, and business calendar runtime
- Backup, restore, PWA, offline, and validation gates

## Not Included

- Backend service
- Cloud sync
- Email, SMS, or push notification delivery
- Automatic financial transaction execution
- Multi-user collaboration
