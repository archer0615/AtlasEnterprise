# Codex Deployment Rules — GitHub Pages

- Atlas v1 deployment is GitHub Pages Static PWA.
- Build with Vite.
- Configure repository base path.
- Use GitHub Actions Pages deployment.
- Use HashRouter.
- Validate manifest, icons, service worker, and offline fallback.
- Fail CI on type, unit-test, build, or PWA validation errors.
- Do not require ASP.NET Core, PostgreSQL, EF Core, Server REST API, Cloud Database, Authentication Server, Multi-device Sync, or always-online backend behavior for deployment.
- Do not commit generated personal backups.
