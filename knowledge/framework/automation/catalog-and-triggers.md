# Automation Catalog and Triggers

## Purpose
This split document isolates automation catalog, trigger matrix, condition matrix, and action matrix from the parent Automation Framework.

## Source
- Parent specification: [Automation Framework](../automation-framework.md)

## Catalog Scope
Automation definitions coordinate scheduled, event-driven, workflow-driven, and rule-driven behavior across Atlas modules.

## Trigger Model
Automation triggers map source events, schedules, workflow states, rule outputs, commands, domain events, and projection changes to action execution.
