# Scheduler Catalog and Timing

## Purpose
This split document isolates scheduler catalog entries, cron strategy, calendar strategy, and scheduling timing behavior from the parent Scheduler Framework.

## Source
- Parent specification: [Scheduler Framework](../scheduler-framework.md)

## Catalog Scope
Schedulers define recurring and calendar-aware triggers for background jobs, workflows, automations, commands, domain events, and operational maintenance.

## Timing Rules
Timing rules include cron strategy, calendar strategy, business-day handling, cutoff behavior, timezone handling, misfire detection, catch-up windows, and concurrency limits.
