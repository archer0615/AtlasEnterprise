# Backup Trigger and Scope

## Purpose
This split document isolates backup automation triggers, backup scope, and eligibility from the parent BackupAutomation specification.

## Source
- Parent specification: [BackupAutomation](../backup-automation.md)

## Trigger Scope
Triggers include scheduled backup, manual backup, pre-migration backup, incident response backup, and retention checkpoint backup.

## Backup Scope
Backup scope includes database state, configuration, document indexes, generated artifacts, audit evidence, and restore metadata where policy allows.
