# Scenario Refresh Trigger and Scope

## Purpose
This split document isolates scenario refresh triggers, source scope, and refresh eligibility from the parent ScenarioRefreshAutomation specification.

## Source
- Parent specification: [ScenarioRefreshAutomation](../scenario-refresh-automation.md)

## Trigger Scope
Triggers include assumption version changes, goal changes, cash flow changes, portfolio changes, loan changes, decision changes, scheduled refresh, and manual refresh.

## Eligibility
Refresh eligibility preserves scenario status, source freshness, household scope, tenant scope, and idempotency.
