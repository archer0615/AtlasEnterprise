# Cleanup Trigger and Scope

## Purpose
This split document isolates cleanup automation triggers, cleanup target scope, and eligibility from the parent CleanupAutomation specification.

## Source
- Parent specification: [CleanupAutomation](../cleanup-automation.md)

## Trigger Scope
Triggers include scheduled cleanup, retention windows, archive expiration, stale cache, stale temporary data, failed job recovery, and manual cleanup requests.

## Cleanup Scope
Cleanup scope includes expired records, temporary artifacts, stale projections, obsolete snapshots, dead-letter retention, and archived data eligible for policy action.
