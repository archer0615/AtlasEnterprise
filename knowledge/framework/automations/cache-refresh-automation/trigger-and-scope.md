# Cache Refresh Trigger and Scope

## Purpose
This split document isolates cache refresh triggers, cache scope, and refresh eligibility from the parent CacheRefreshAutomation specification.

## Source
- Parent specification: [CacheRefreshAutomation](../cache-refresh-automation.md)

## Trigger Scope
Triggers include source data changes, projection updates, stale cache windows, manual refresh, scheduled refresh, and dependency invalidation.

## Cache Scope
Cache scope includes dashboard, report, search, scenario, recommendation, decision, goal, portfolio, and notification projections.
