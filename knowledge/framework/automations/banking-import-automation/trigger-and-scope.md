# Banking Import Trigger and Scope

## Purpose
This split document isolates banking import triggers, account scope, source eligibility, and import boundaries from the parent BankingImportAutomation specification.

## Source
- Parent specification: [BankingImportAutomation](../banking-import-automation.md)

## Trigger Scope
Triggers include scheduled banking import, manual import, external webhook, credential refresh, failed import recovery, and reconciliation refresh.

## Import Scope
Import scope includes accounts, balances, transactions, fees, interest, currency, and source timestamps under tenant and household boundaries.
