# Portfolio Performance Attribution and Reporting

## Purpose

This split document isolates attribution and reporting behavior from the larger Portfolio Performance Framework.

## Source

- Parent specification: [Portfolio Performance Framework](../portfolio-performance-framework.md)

## Attribution Rules

- Attribution should explain performance drivers without changing Portfolio source state.
- Attribution should distinguish allocation, selection, cash flow, and market movement effects.
- Attribution calculations must remain reproducible for the same input period.

## Reporting Rules

- Dashboard reporting may summarize performance and drawdown.
- Reports must include benchmark, period, method, and calculation version.
- Stress scenario reports must identify assumptions and warning references.
