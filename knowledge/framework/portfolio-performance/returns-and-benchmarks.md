# Portfolio Performance Returns and Benchmarks

## Purpose

This split document isolates return and benchmark rules from the larger Portfolio Performance Framework.

## Source

- Parent specification: [Portfolio Performance Framework](../portfolio-performance-framework.md)

## Return Rules

- Time-weighted return is the default performance method.
- Money-weighted return may be used when investor cash flows matter.
- Return calculations must preserve formula version and input period.
- Performance outputs must remain explainable for dashboard and decision use.

## Benchmark Rules

- Benchmarks must be explicit and versioned.
- Benchmark comparison must separate investment skill from market movement.
- Benchmark outputs should preserve assumptions and reporting period.
