# Goal Funding Targets and Contributions

## Purpose

This split document isolates target and contribution behavior from the larger Goal Funding Framework.

## Source

- Parent specification: [Goal Funding Framework](../goal-funding-framework.md)

## Target Rules

- Goal funding targets must preserve currency, target date, and required amount.
- Target calculations must remain explainable and formula-versioned.
- Target changes must remain auditable when used for recommendations.

## Contribution Rules

- Contribution plans must identify required monthly contribution.
- Contribution feasibility must consider cashflow safety and reserve constraints.
- Contribution changes should preserve priority and scenario context.
