# Goal Funding Prioritization and Reallocation

## Purpose

This split document isolates goal funding prioritization and reallocation behavior.

## Source

- Parent specification: [Goal Funding Framework](../goal-funding-framework.md)

## Prioritization Rules

- Goal priority must preserve existing goal governance and conflict rules.
- Funding priority should account for urgency, feasibility, and risk.
- Priority changes must be explainable when surfaced as recommendations.

## Reallocation Rules

- Reallocation must preserve cashflow and reserve constraints.
- Reallocation scenarios must compare baseline and alternative outcomes.
- Reallocation recommendations must include warning references.
