# Goal Lifecycle Automation and Execution

## Purpose
This split document isolates Goal lifecycle automation, commands, events, workflow integration, recovery, and reporting behavior from the parent Goal Lifecycle Management specification.

## Source
- Parent specification: [Goal Lifecycle Management](../goal-lifecycle-management.md)

## Automation
Automation includes automatic activation, completion, archive, reminder, review, escalation, recovery, and cleanup.

## Commands
Commands include create, update, activate, pause, resume, block, unblock, complete, cancel, archive, restore, delete, merge, split, clone, convert to template, create snapshot, expire, report, and recalculate lifecycle state.
