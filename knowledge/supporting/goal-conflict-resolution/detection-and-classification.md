# Goal Conflict Detection and Classification

## Purpose
This split document isolates goal conflict detection inputs, classification, severity, and impact analysis from the parent Goal Conflict Resolution specification.

## Source
- Parent specification: [Goal Conflict Resolution](../goal-conflict-resolution.md)

## Detection Scope
Goal conflict detection compares priority, timing, funding, cash flow, dependency, risk, household preference, decision outcome, recommendation impact, and scenario results.

## Classification
Conflicts are classified by resource conflict, timing conflict, priority conflict, dependency conflict, risk conflict, cash-flow conflict, and recommendation conflict.
