# Goal Conflict Resolution Workflow

## Purpose
This split document isolates conflict resolution workflow, decision support, recommendation mapping, and outcome handling from the parent Goal Conflict Resolution specification.

## Source
- Parent specification: [Goal Conflict Resolution](../goal-conflict-resolution.md)

## Workflow
Resolution workflow identifies conflict evidence, ranks severity, evaluates alternatives, maps recommendations, records user decision, updates related plans, and preserves audit history.

## Outputs
Outputs include conflict summary, severity, affected goals, proposed resolution, trade-off explanation, recommendation references, and follow-up actions.
