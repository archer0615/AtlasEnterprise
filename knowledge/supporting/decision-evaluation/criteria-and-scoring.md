# Decision Evaluation Criteria and Scoring

## Purpose
This split document isolates decision evaluation criteria, scoring model, thresholds, weights, and result interpretation from the parent Decision Evaluation specification.

## Source
- Parent specification: [Decision Evaluation](../decision-evaluation.md)

## Criteria
Criteria include financial impact, cash flow impact, risk, goal alignment, feasibility, urgency, confidence, scenario performance, recommendation quality, and execution readiness.

## Scoring
Scoring preserves weighted metrics, normalized scores, threshold ranges, rule outputs, penalty factors, bonus factors, and explainability evidence.
