# Decision Evaluation Execution and Evidence

## Purpose
This split document isolates decision evaluation execution pipeline, evidence capture, commands, events, APIs, and projection outputs from the parent Decision Evaluation specification.

## Source
- Parent specification: [Decision Evaluation](../decision-evaluation.md)

## Execution
Execution collects inputs, validates assumptions, applies rules, calculates scores, compares scenarios, records evidence, emits evaluation results, and refreshes dashboard projections.

## Evidence
Evidence preserves source references, metric values, rule versions, scenario versions, assumption versions, calculation timestamp, and evaluator context.
