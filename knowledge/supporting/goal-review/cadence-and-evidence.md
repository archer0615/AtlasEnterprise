# Goal Review Cadence and Evidence

## Purpose
This split document isolates Goal Review cadence, review types, source evidence, criteria, and metrics from the parent Goal Review specification.

## Source
- Parent specification: [Goal Review](../goal-review.md)

## Review Lifecycle
Goal Review begins when a goal reaches scheduled, manual, automatic, exception, risk, financial, or completion review conditions. It continues through preparation, data collection, evaluation, scoring, decision, action items, approval, and completion.

## Review Frequency
Review frequency includes monthly, quarterly, annual, manual, automatic, exception, risk, financial, and goal completion review cadence.

## Evidence Scope
Goal Review evidence includes Goal, Milestone, Task, Decision, Recommendation, Scenario, Portfolio, CashFlow, Notification, and User signals.
