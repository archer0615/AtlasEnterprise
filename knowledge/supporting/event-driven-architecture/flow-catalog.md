# Event-Driven Flow Catalog

## Purpose
This split document isolates the complete event flow catalog from the parent Event Driven Architecture specification.

## Source
- Parent specification: [Event Driven Architecture](../event-driven-architecture.md)

## Event Flow Catalog
The flow catalog includes salary, expense, portfolio, security, loan, home, policy, premium, retirement, scenario, recommendation, decision, rule, constraint, score, snapshot, and replay event flows.

## Flow Contract
Each flow preserves publisher, subscriber, trigger, payload, ordering, retry behavior, idempotency key, projection updates, audit requirements, and failure behavior.
