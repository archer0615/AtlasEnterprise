# Goal Insights Generation

## Purpose
This split document isolates Goal Insight generation, detection rules, scoring, ranking, and evidence collection from the parent Goal Insights specification.

## Source
- Parent specification: [Goal Insights](../goal-insights.md)

## Generation Scope
Goal Insights consume Goal, Goal Progress, Goal Metrics, Goal Review, Goal Analytics, Recommendation, Decision, Scenario, Portfolio, CashFlow, Risk, and User signals.

## Detection
Detection covers progress risk, budget pressure, schedule delay, dependency blockage, priority drift, review gaps, recommendation non-adoption, and forecast variance.
