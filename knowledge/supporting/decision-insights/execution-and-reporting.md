# Decision Insights Execution and Reporting

## Purpose
This split document isolates Decision Insight commands, APIs, DTOs, persistence, reporting, security, audit, performance, and testing from the parent Decision Insights specification.

## Source
- Parent specification: [Decision Insights](../decision-insights.md)

## Commands
Commands include CreateInsight, UpdateInsight, RefreshInsight, GenerateInsight, DismissInsight, ResolveInsight, ArchiveInsight, RestoreInsight, DeleteInsight, and GenerateInsightReport.

## Reporting
Reporting exposes insight summary, detail, search, report projection, dashboard aggregation, evidence summaries, recommendations, and status history under permission-aware masking.

## Audit
Audit covers insight history, evidence history, resolution history, recommendation history, access history, source snapshots, rule versions, and operator actions.

## Testing
Testing covers unit, integration, insight, detection, evidence, performance, concurrency, stress, forecast, API, cache, and export behavior.
