# Decision Analytics Indicators

## Purpose
This split document isolates Decision Analytics categories, indicators, formulas, thresholds, and dashboard metric usage from the parent Decision Analytics specification.

## Source
- Parent specification: [Decision Analytics](../decision-analytics.md)

## Indicator Scope
Indicators cover decision completion, acceptance, rejection, execution, success, accuracy, recommendation stability, scenario adoption, reversal frequency, and time-to-decision metrics.
