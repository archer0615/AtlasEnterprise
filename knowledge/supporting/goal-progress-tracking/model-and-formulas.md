# Goal Progress Model and Formulas

## Purpose
This split document isolates progress dimensions, calculation formulas, update frequency, and consistency requirements from the parent Goal Progress Tracking specification.

## Source
- Parent specification: [Goal Progress Tracking](../goal-progress-tracking.md)

## Progress Dimensions
- Physical Progress
- Financial Progress
- Time Progress
- Milestone Progress
- Dependency Progress
- Risk Progress
- Confidence Score
- Completion Score
- Health Score
- Overall Progress

## Calculation Formulas
- Weighted Progress
- Milestone Contribution
- Financial Contribution
- Elapsed Time %
- Remaining Time %
- Schedule Variance
- Budget Variance
- Forecast Completion
- Expected Completion Date
- Progress Confidence
- Goal Health Score
- Risk Adjusted Progress
- Composite Progress Score

## Update Rules
Progress updates can be manual, automatic, event driven, batch, scheduled, scenario simulation based, recommendation driven, decision driven, rollback based, or recalculated.
