# Recommendation Evaluation Scoring and Constraints

## Purpose
This split document isolates recommendation evaluation dimensions, scoring model, ranking behavior, and constraint evaluation from the parent Recommendation Evaluation specification.

## Source
- Parent specification: [Recommendation Evaluation](../recommendation-evaluation.md)

## Evaluation Dimensions
- Goal Alignment
- Financial Benefit
- Cash Flow Impact
- Portfolio Impact
- Risk Level
- Priority
- Urgency
- Feasibility
- Execution Complexity
- Dependency
- Expected ROI
- Expected Value
- Confidence Score
- Explainability Score
- User Preference Alignment
- Time Sensitivity
- Business Value

## Scoring Model
Scoring includes weighted score, composite score, normalized score, confidence score, penalty rules, bonus rules, ranking rules, and tie-breaking rules.

## Constraint Evaluation
Constraints include hard, soft, budget, risk, portfolio, cash flow, goal, scenario, compliance, and user preference constraints.
