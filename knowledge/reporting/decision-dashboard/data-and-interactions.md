# Decision Dashboard Data and Interactions

## Purpose
This split document isolates decision dashboard data sources, projections, interaction behavior, and refresh rules from the parent Decision Dashboard specification.

## Source
- Parent specification: [Decision Dashboard](../decision-dashboard.md)

## Data Sources
Dashboard data consumes Decision, Recommendation, Scenario, Decision History, Decision Metrics, Decision Audit, Decision Insights, Portfolio, CashFlow, Goal, and User preferences.

## Interactions
Interactions include filtering, search, sorting, scenario switching, decision selection, action triggering, refresh, and empty or error states.
