# Decision Dashboard Metrics and Views

## Purpose
This split document isolates dashboard-visible decision metrics, view composition, filtering, and status summaries from the parent Decision Dashboard specification.

## Source
- Parent specification: [Decision Dashboard](../decision-dashboard.md)

## Metrics
Decision dashboard metrics include decision count, status distribution, acceptance rate, execution rate, stale decisions, recommendation stability, scenario comparison, and outcome success.

## Views
Views include dashboard summary, decision timeline, recommendation list, scenario comparison, action queue, warning panel, and historical trend.
