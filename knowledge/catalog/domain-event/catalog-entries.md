# Domain Event Catalog Entries

## Purpose
This split document isolates domain event definitions from the parent Domain Event Catalog.

## Source
- Parent specification: [Domain Event Catalog](../domain-event-catalog.md)

## Event Groups
Domain events cover cash flow, investment, loan, home, insurance, retirement, decision, rule, and system event groups.

## Entry Contract
Each event entry preserves event code, name, aggregate owner, publisher, payload, trigger command, consumers, ordering, idempotency, audit classification, and replay behavior.
