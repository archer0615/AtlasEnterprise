# Enumeration Catalog Entries

## Purpose
This split document isolates enumeration definitions from the parent Enumeration Catalog.

## Source
- Parent specification: [Enumeration Catalog](../enumeration-catalog.md)

## Catalog Scope
Enumerations define canonical status, type, category, priority, severity, lifecycle, permission, module, and domain option sets used across Atlas.

## Entry Contract
Each enumeration entry preserves enum name, values, descriptions, owning module, API serialization, database mapping, default behavior, and deprecation rules.
