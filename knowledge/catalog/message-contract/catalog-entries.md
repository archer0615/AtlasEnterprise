# Message Contract Catalog Entries

## Purpose
This split document isolates message contract definitions from the parent Message Contract Catalog.

## Source
- Parent specification: [Message Contract Catalog](../message-contract-catalog.md)

## Catalog Scope
Message contracts define event, command, integration, notification, workflow, automation, scheduler, and reporting payloads exchanged between Atlas modules and external systems.

## Contract Requirements
Each contract preserves name, producer, consumer, schema, version, idempotency key, correlation identifiers, causation identifiers, tenant scope, permission scope, and audit classification.
