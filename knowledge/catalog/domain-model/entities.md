# Domain Model Entities

## Purpose
This split document isolates domain model entity definitions from the parent Domain Model Catalog.

## Source
- Parent specification: [Domain Model Catalog](../domain-model-catalog.md)

## Entity Scope
Domain model entities define identity, ownership, lifecycle, attributes, commands, events, repositories, invariants, persistence mapping, and API exposure for Atlas business concepts.

## Catalog Usage
Each entity entry must preserve its bounded context, aggregate ownership, service ownership, validation rules, audit requirements, security classification, and consistency boundaries.
