# Entity Catalog Entries

## Purpose
This split document isolates entity catalog definitions from the parent Entity Catalog.

## Source
- Parent specification: [Entity Catalog](../entity-catalog.md)

## Entity Entries
Entity catalog entries include User, Household, Asset, Liability, Goal, Portfolio, Holding, Property, Mortgage, Scenario, Decision, Recommendation, Notification, Policy, and Configuration.

## Entry Contract
Each entity entry preserves identity, aggregate owner, lifecycle, properties, commands, events, repository, API, security, audit, and persistence mapping.
