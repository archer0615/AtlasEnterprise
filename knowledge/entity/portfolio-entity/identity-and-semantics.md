# Portfolio Identity and Semantics

## Purpose
This split document isolates Portfolio identity, aggregate boundary, lifecycle, ownership, relationships, properties, portfolio semantics, ownership model, asset association model, validation, and business rules from the parent Portfolio Entity Specification.

## Source
- Parent specification: [Portfolio Entity Specification](../Portfolio.md)

## Identity
Portfolio represents an investment container with owner, base currency, lifecycle state, risk context, associated assets, and valuation behavior.

## Semantics
Portfolio semantics cover target allocation, current allocation, associated positions, valuation summary, lifecycle status, owner visibility, and dashboard projection.
